# 图片
格式 全称 浏览器支持 透明 优点 缺点
PNG Portable Network Graphics 主流都支持 ✅ 高质量 无损压缩 大体积
JP[E]G Joint Photographic Experts Group 支持广泛 ❌ 有损，体积小，浏览器、操作系统支持广泛 有损
SVG 主流都支持 ✅
GIF Graphics Interchange Format 主流都支持 ✅ 动画，体积小， 8-bit约束，清晰度受限制
WebP 新版支持 ✅ 高性能
AVIF AV1 Image File Format 新版支持 ✅ 超高性能,android12 支持 转码时间长
ICO icon 主流都支持 ✅
APNG Animated Portable Network Graphics 新版支持 ✅ 更高性能，动态，向下兼容PNG
PDF Portable Document Format 可查看，不可内嵌 ❌
RAW 不支持 ❌ 超高质量 大体积
TIFF Tagged Image File Format 需要扩展支持 高质量，印刷 大体积
HEIC 新版支持 ✅ 更高性能 mac支持
HEIF High Efficiency Image File Format 主流都不支持 ❌ 高性能 仅手机支持
EPS Encapsulated PostScript 不支持
